# Deep Study AI

## Description
Deep Study AI est une application qui transforme vos cours en résumés, fiches de révision, quiz et mindmaps grâce à l'IA.

## Installation
1. Cloner le dépôt
2. Installer les dépendances :
   - Backend : `pip install -r backend/requirements.txt`
   - Frontend : `npm install`
3. Configurer Firebase (voir `firebase-config.json`).

## Lancement
- Backend : `uvicorn backend.main:app --reload`
- Frontend : `npm start`
